## Key Learnings
- **内联样式无法做响应式**：倒计时原来是写死的 inline style，手机端才会显得大。改成 CSS 类 + `@media(max-width:640px)` 才能自适应。
- **微信群进群硬限制**：群二维码 7 天过期；群满 100 人后二维码彻底失效，新人只能靠群成员邀请。→ 大群（新生群）必须走「人工拉」。
- **命令行长命令别在 Claude 输入框换行**：`claude mcp add playwright -s user -- npx @playwright/mcp@latest` 被换行拆断，`@playwright/mcp@latest` 掉到第二行被 bash 当独立命令执行，结果只存了半条（command=npx、args=[]）。
- **Windows 下 npx 起 MCP 可能要 `cmd /c` 包裹**（历史坑，新版本可能已修，失败再改）。
- searchIndex 手写下标锚点脆弱（CLAUDE.md 早有警告），本次体检发现「怎么选课」错指 pitfall-1-0（四六级），已删。
