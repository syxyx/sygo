## Decisions Made
1. **移动端 + 视觉交互优先**：用户先做体验，暂不碰转化
2. **硬信息不猜、以官方为准**：学费/校区/报到日期这类错一个字砸招牌的信息，只用核实过的官方数据或诚实指路官方（招生办 0739-5432591）
3. **用本地 curl 绕过官网抓取限制**：WebFetch 走美国服务器连不上国内站，Bash curl 走用户网络能连
4. **临床医学校区标注"以通知书为准"**：官网显示七里坪+西湖都有，不硬下结论
5. **下一步方向**：做「专业→校区」查询页，或补"报到前"高频缺口

## Key Learnings
- **WebFetch/WebSearch 连不上任何国内网站**（.edu.cn/.net、百度、阳光高考全拦）——浏览走 Anthropic 美国服务器，非用户电脑
- **Bash/curl 在用户 Windows 本地跑，走用户网络**——能访问官网，这是抓国内数据的通道
- **Python 是 Windows 商店空壳（exit 49）**，改用 **Node v24**；node 把 `/tmp` 当 `D:\tmp`，临时文件放工作目录
- **招生网接口**：`POST http://zsw.hnsyu.edu.cn/api.php/cms/getschoollist`（空 body）→ 双层编码 JSON（外层 data 是字符串需再 parse）→ 20 学院+专业+电话
- **校区从各学院官网页脚地址提取**：`http://www1.hnsyu.net/<代码>/`（如 lcyxx=普爱, yxx=药学, hlx=护理）
- **内容评估**：网站内容 8 分（真有用），变现漏斗 4 分，截流基建 3 分；作为"查问题工具"能超越抖音学长（结构化可搜索），但缺算法触达+真人互动
